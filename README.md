Desenvolvimento de um sistema web de agendamento voltado para fotógrafos, utilizando Python (Flask) com integração a banco de dados PostgreSQL via SQLAlchemy.
O projeto permite o cadastro de agendamentos com validação de conflitos de horários, persistência dos dados em banco relacional e estrutura MVC com templates HTML e CSS.

Tecnologias e conceitos aplicados:

Python & Flask

PostgreSQL + pgAdmin

SQLAlchemy (ORM)

Validação de horários e regras de negócio

Estruturação de projeto web (backend + frontend)

Projeto focado em boas práticas e aprendizado prático de backend para aplicações reais.
